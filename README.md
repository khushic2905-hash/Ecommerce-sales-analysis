# 📊 E-commerce Sales Analysis using SQL & Power BI

## 📌 Project Overview
This project analyzes e-commerce sales data using SQL Server and Power BI. The dataset was cleaned, transformed, and analyzed to uncover key business insights. An interactive dashboard was created to help visualize sales performance and support data-driven decision-making.

---

## 🎯 Objectives
- Clean and prepare the sales dataset.
- Analyze sales performance using SQL.
- Identify top-performing products and categories.
- Measure key business KPIs.
- Build an interactive Power BI dashboard.

---

## 🛠️ Tools & Technologies
- SQL Server
- SQL
- Power BI
- CSV Dataset

---

## 📂 Project Workflow
1. Imported the dataset into SQL Server.
2. Performed data cleaning:
   - Checked for missing values
   - Removed duplicates
   - Verified data types
3. Created calculated columns (Revenue).
4. Solved business problems using SQL queries.
5. Built an interactive Power BI dashboard.

---

## 📊 Key Performance Indicators (KPIs)
- Total Revenue
- Total Orders
- Total Quantity Sold
- Average Revenue per Order

---

## 📈 Dashboard Insights
- Identified the highest revenue-generating product categories.
- Compared sales performance across different regions.
- Analyzed order quantity and revenue trends.
- Evaluated overall business performance through KPIs.
